# Postman API Testing Project

## Project Overview
This project demonstrates REST API testing using Postman and ReqRes API.

## APIs Tested
- GET Users
- GET Single User
- POST Create User
- PUT Update User
- DELETE User
- Login API

## Testing Performed
- Status code validation
- Response validation
- CRUD testing
- Negative testing
- Automated test scripts

## Tools Used
- Postman
- ReqRes API
- GitHub

## Features
- Environment Variables
- API Authentication
- Collection Runner
- Automated Assertions
